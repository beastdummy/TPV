Imágenes de la web (Vite / public)

Coloca aquí archivos estáticos. Se sirven desde la raíz del sitio, por ejemplo:
  /images/tu-archivo.jpg

En uso (site-assets.ts):
  hero.jpg       — Portada (hero)
  demo.jpg       — Página /demo
  showcase.jpg   — Franja en home (features → precios)

Opcional:
  logo.svg / logo.png  — Cabecera / pie
  og.png               — 1200×630 redes (meta futuro)

Formatos: WebP o AVIF para fotos, SVG para logotipos.
Optimiza el peso antes de subir (Squoosh, etc.).
